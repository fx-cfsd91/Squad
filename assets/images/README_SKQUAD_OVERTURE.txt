Fichier attendu pour la page `overture`.

Placez l'image fournie (logo) au chemin exact suivant dans le projet :

  assets/images/squad_logo.png

Format recommandé : PNG ou WEBP; taille conseillée : largeur 1600px (ou équivalent) et fond transparent si possible.

Après ajout, rechargez Metro / Expo pour voir la page :

  npx expo start -c

Accès rapide depuis l'app (si vous utilisez l'UI) :
  route /ouverture

Si vous voulez que je copie l'image jointe automatiquement dans le projet, uploadez-la dans la racine du workspace ou donnez-moi l'emplacement exact. (Je ne peux pas lire les pièces jointes depuis la conversation pour écrire automatiquement le binaire sans que vous me confirmiez où le placer.)
